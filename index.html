Description of Application

The application gets all public repositories from GitHub using GitHub API


I used "GET /repositories" with since string that define last repository you have seen


For pagnitation I get next link form response header


In this case "page" and "per_page" dosen't work, so for show 20 items per page I have to use my idea. In first step I get all repositories from begining link (since=0) and put it to array. Then I show elements ofr this array form 1 to 20. When all elements form array are showed then I get next url.

Try it!
